# 🧠 MindStream - La evolución de tus pensamientos

App móvil de diario emocional con transcripción de voz, diseñada para registrar pensamientos durante el día y exportarlos a PDF.

## ✨ Funcionalidades

- 📝 **Registro de pensamientos** con entrada de texto y voz
- 🏷️ **8 categorías emocionales**: Tristeza, Alegría, Recuerdo, Ansiedad, Gratitud, Reflexión, Miedo, Esperanza
- 📊 **Dashboard** con estadísticas diarias
- 🔍 **Búsqueda multidimensional**: por texto, fecha, día, hora y categoría
- 📄 **Exportación a PDF** con diseño profesional y nombre personalizado
- 👤 **Perfil de usuario** con foto y nombre
- 🧘 **Ejercicio de respiración 4-7-8**
- 💬 **Afirmaciones diarias** personalizables
- 📱 **Interfaz optimizada** para dispositivos móviles

## 🛠️ Tecnologías

- **Expo SDK 54** + React Native
- **TypeScript**
- **react-native-paper** (UI)
- **expo-sqlite** (base de datos local)
- **expo-print** + **expo-sharing** (exportación PDF)

## 🚀 Instalación

```bash
# Clonar repositorio
git clone https://github.com/juan.reyes/mindstream.git
cd mindstream

# Instalar dependencias
npm install

# Iniciar con Expo Go
